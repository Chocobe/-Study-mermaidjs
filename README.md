# mermaid.js 스터디

## 개요

### 스터디 목적

* 개발 문서의 품질 향상
* 기존 다이어그램 툴 대비 빠른 작성과 쉬운 수정 가능
* 설계의 문서화를 통해서 개발 단계에서 흔들리지 않는 구현 기대
* 설계 공유
* `mermaid.js` 에서 제공하는 다양한 다이어그램 사용 가능
